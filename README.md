# BasicVA
Simple voice assistant that utilizes several python libraries such as pyttsx3 and SpeechRecognition while also utilizing AI capabilities from Groq API.

# Set up before using the program
1. Add a creds.py and reference it to your .gitignore.
2. Add your free groq API key with the format 
    API_KEY = ""
    Get your API KEY from this URL - https://console.groq.com/keys

# How to use the program?
1. Speak your prompt carefully while it prints "Listening..."
2. After you have finished your speech prompt, press any key.
3. Wait for the results
